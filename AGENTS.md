# AGENTS.md

## Project Overview

This project is a workout tracking web application built with Google Apps Script.

Frontend:

* HTML
* JavaScript

Backend:

* Google Apps Script

Storage:

* Google Sheets

## Main Entities

### Workouts

Workout sessions.

### WorkoutSets

Sets performed during workouts.

### Exercises

Exercise catalog.
