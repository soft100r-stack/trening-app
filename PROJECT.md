# Trening App

## Назначение

Веб-приложение для учета тренировок и подходов.

## Текущий стек

* Google Apps Script
* HTML
* JavaScript
* Google Sheets

## Основные сущности

### Exercises

Список упражнений.

### Workouts

Тренировки пользователя.

### WorkoutSets

Подходы внутри тренировки.

## Уже работает

* создание тренировок;
* добавление упражнений;
* добавление подходов;
* хранение данных в Google Sheets;
* веб-интерфейс через Apps Script.

## Ближайшие планы

1. Улучшение интерфейса.
2. История тренировок.
3. Личные рекорды.
4. Аналитика прогресса.
5. Подготовка к масштабированию.
6. Возможный переход на отдельный backend в будущем.

## Правила разработки

* Не ломать существующую функциональность.
* Изменения делать небольшими шагами.
* Сохранять совместимость с текущими данными.
* Предпочитать простые решения.
