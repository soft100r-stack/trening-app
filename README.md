# Trening App

Приложение для учета тренировок.

Текущие возможности:

* создание тренировок;
* добавление упражнений;
* учет подходов и повторений;
* хранение данных через Google Apps Script.

Планы развития:

* история тренировок;
* личные рекорды;
* аналитика прогресса;
* авторизация пользователей;
* мобильный интерфейс.

Технологии:

* Google Apps Script
* HTML
* JavaScript
