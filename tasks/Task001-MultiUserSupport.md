# Task 001 - Multi User Support

Цель:
Добавить поддержку нескольких пользователей без изменения существующих данных.

Требования:

1. Создать сущность Users.

Поля:

* UserID
* Email
* Name
* CreatedAt

2. Добавить поле UserID в таблицу Workouts.

3. При первом входе пользователь может зарегистрироваться через email и имя.

4. Если пользователь уже существует, загружать только его тренировки.

5. Все новые тренировки должны сохраняться с UserID текущего пользователя.

6. Не изменять структуру WorkoutSets.

7. Сохранить совместимость со всеми существующими тренировками.
